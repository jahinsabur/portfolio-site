CLASH DISPLAY FONT SETUP
========================

Download Clash Display:
https://www.fontshare.com/fonts/clash-display

1. Click "Download family"
2. Extract the ZIP
3. Find: ClashDisplay-Variable.woff2 (in Fonts/WEB/ folder)
4. Place it here: ClashDisplay-Variable.woff2

The site will use system fonts as fallback until you add this file.
